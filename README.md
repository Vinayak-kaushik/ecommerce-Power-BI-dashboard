# 📊 E-commerce Sales Dashboard (Power BI)

## 📌 Project Overview
This project analyzes e-commerce sales data and presents insights using an interactive Power BI dashboard.

## 📈 Key Insights
- Clothing category has highest sales
- Majority of orders are shipped successfully
- Top sales from Delhi and Karnataka
- Size XL and NA are most preferred

## 🛠 Tools Used
- Power BI
- Excel
- DAX
